Audit every file and symbol in this codebase for naming consistency. Apply these rules:

FILES:
  - React components → PascalCase.jsx (UserCard.jsx, not usercard.jsx or user-card.jsx)

  - Sections → PascalCase ending in "Section" (HeroSection.jsx)
  - Context → PascalCase ending in "Context" (AuthContext.jsx)
  - Constants → SCREAMING_SNAKE_CASE inside the file, camelCase filename (constants.js)

  - React components (functions) → PascalCase
  - All other functions → camelCase

  - Event handlers → prefix with "handle" (handleSubmit, handleClick)
  - Constants → SCREAMING_SNAKE_CASE

  - Tailwind → no change needed
  - CSS variables → kebab-case (--color-primary)
