RE-RENDERS:
  - Find components that re-render on every parent render despite same props
    Fix: wrap in React.memo()
  - Find expensive calculations done inline in render
    Fix: wrap in useMemo()
  - Find functions recreated on every render and passed as props
    Fix: wrap in useCallback()

BUNDLE SIZE:

    Fix: use React.lazy() + Suspense for route-level code splitting

    Fix: add loading="lazy" to <img> tags

  - Find setState calls inside loops
  - Find multiple sequential setState calls that could be batched
  - Find state that's actually derivable from other state
